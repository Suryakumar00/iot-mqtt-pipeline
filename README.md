# IoT MQTT Pipeline

## 🚀 Project Overview
This project establishes an end-to-end IoT data pipeline, integrating embedded hardware with a modern backend. It features a simulated ESP32 device that transmits environmental sensor data to a concurrent Golang backend service via the MQTT protocol.

## 🛠️ Technical Stack
* **Firmware:** ESP32 (simulated via Wokwi)
* **Communication:** MQTT (broker.emqx.io)
* **Backend:** Golang
